Introdução ao HTML

O que é HTML?

O HTML (Linguagem de Marcação de Hipertexto) é a espinha dorsal da web. Ele define a estrutura dos conteúdos apresentados nos navegadores.

Estrutura Básica

A estrutura de um documento HTML é composta por elementos que definem o título, corpo, cabeçalho, parágrafos, listas e outros.

Por exemplo, podemos destacar uma parte importante do texto para chamar a atenção do leitor.

Em notas de rodapé ou observações, é comum utilizar a tag este texto é uma observação adicional para diminuir a ênfase.

Também é comum o uso de itálico para enfatizar termos estrangeiros e sublinhar alguns conceitos chave.

A tag font pode ser utilizada para modificar a cor e o tipo da fonte, embora seu uso esteja em desuso em favor do CSS.

Algumas vezes, é necessário demarcar informações que não são mais pertinentes dentro do conteúdo.
